## 🎬 What it does

Manage everything from a browser:

- **Live player list** with ping, health, vehicle, full identifiers, IP
- **Watch any player's screen** straight from the dashboard (with recording)
- **Live map** showing player positions and kill heatmap
- **Player database**: every account ever connected, grouped by license
- **Inventory search**: find who has what, across the whole server
- **Bans**: create, edit, lift, with full audit
- **Vehicles**: every parked / stored / impounded car, with repair / refuel / force-store
- **Economy tracker** with auto-alerts for dupes and suspicious transfers
- **Resource manager**: start / stop / restart from the browser
- **FiveM console** live-tail and execute commands
- **Audit log**: every admin action recorded forever
- **Discord OAuth login** and Discord role sync
- **Granular permissions**: 40+ flags, build custom roles
- **Queue and whitelist** management

Plus chat history, kill log, property management, file storage, phone health checks, entity counters, and more.

## ⚡ Why it is not another admin menu

- Runs on **one port**, behind nginx + SSL. Not an in-game NUI.
- Mobile friendly: manage your server from your phone.
- Granular permissions, not "admin / not admin".
- Audit log is append-only. Receipts for every action.
- Works with `ox_inventory`, `jg-advancedgarages`, `lb-phone` out of the box.
- Captures default Cfx `chatMessage` event, works with any chat resource.
- Auto-detects QBox / QBCore / ESX.

## ⚙️ Install

1. Download release ZIP from Tebex.
2. Upload to Linux server, unzip into `/home/fivem`.
3. `bash deploy/setup-linux.sh` and answer prompts.

The script installs Node.js, nginx, SSL, dependencies, and runs everything as a systemd service. About 5 minutes start to finish.

Then drop `nexora-dashboard` (Lua resource) into your FiveM resources, set two config values, restart, done.

Manual install instructions in the docs for people who want full control over nginx.

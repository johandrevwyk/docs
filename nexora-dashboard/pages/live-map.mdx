Route: `/livemap`
Sidebar group: Overview
Permission: `livemap.view`

## What it does

A top-down heatmap of the Los Santos map showing where players are right now and where deaths recently happened.

## What you see

- **Player dots**, live positions of online players.
- **Heatmap layer**, color intensity shows player density / kill hotspots.
- **Toggle layers**, players, kills, all.

## Common actions

- **Hover dot** → see player name + character.
- **Click dot** → open their profile.
- **Toggle heatmap** → switch between player density and kill density.

## Notes

Position data updates every few seconds. The exact interval is set on [Settings](settings). Kill data covers the last few hours by default; tune retention on [Settings](settings) too.

A web dashboard for your FiveM server. Manage players, vehicles, money, bans, resources and more from your browser, no game client required.

Works with **QBox**, **QBCore** and **ESX**. One install. One port. Real-time updates.

## What is in these docs

- **Getting Started**, install the dashboard on your server (auto script or manual).
- **Pages**, what every screen in the dashboard does.
- **Lua Resource**, drop-in resource that connects your game server to the dashboard.
- **Troubleshooting**, fixes for common problems.

## Need help?

Open the in-app **Help** menu (sidebar bottom-left) for Discord + docs links. If your dashboard is broken before first login, jump straight to [troubleshooting.md](troubleshooting).

## FAQ

**Does it work without a domain?**
You need a domain for SSL. Without HTTPS, the FiveM CEF screen-stream feature breaks.

**Will it slow my game server down?**
No. The Lua resource is lightweight and runs at about 2-second poll intervals by default (tunable).

**Can I host the dashboard on the same VPS as my game server?**
Yes, as long as you have around 1 GB free RAM.

**Can I modify the source?**
No. This is a closed-source paid product. Bug reports and feature requests go through Discord and ship as free updates.

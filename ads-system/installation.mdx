### ⚡ Install (under 60 seconds)

1. Drop `ads-system` into `resources/`.
2. `ensure ads-system` in `server.cfg` after QB-Core / oxmysql / ox\_lib.
3. SQL table auto-creates on first start.
4. Edit `shared/config.lua` — set webhook, currency symbol, cooldown.
5. Edit `shared/adtypes.lua` — add your server's ad types.

That's it. Players type `/ads`.
